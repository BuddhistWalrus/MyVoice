To get the app running:

Make sure you have the Postgres app installed locally. See http://postgresapp.com/ to download.

1. Clone the repo
2. Run 'brew install postgresql'
3. Run 'bundle install'
4. Run 'bin/rake db:create db:migrate'
5. Run 'foreman start' to get the app running. Default port is localhost:5000
